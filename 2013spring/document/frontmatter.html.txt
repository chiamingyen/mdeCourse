寫在前頭
++++++++++++

目前這份手冊只是一個使用 Leo Editor 的開始, 大多數的內容僅為素材與基本架構, 並非完整的參考資料.

由於機械設計是一個牽涉許多溝通與資料的過程, 身處其中的工程師若能夠利用程式方法來產出並管理過程中所衍生的資料, 將有助於提升本身與團隊的工作效益.

採用 Leo Editor、Sphinx 與 MiKTeX 進行文書處理雖然有許多好處, 但是 html 中文內容的分詞與搜尋以及 PDF 的排版, 仍然存在不少問題, 現階段在套用 Leo Editor 手冊編輯模式的同時, 也必須設法找出最適合機械設計流程中管理程式與手冊的最佳模式.

與本手冊相關的資料可以同步由下列 github 網址下載:

https://github.com/chiamingyen/mdeCourse
